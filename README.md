SuperMart Analytics Capstone
Project Overview

This project analyzes sales, customers, products, orders, and employee performance for a fictional retail company, SuperMart, using PostgreSQL.

The objective was to answer business questions related to revenue, customer behavior, product performance, and employee productivity through SQL-based analysis.
